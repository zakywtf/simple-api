# Installation:

Dev Enviroments:

```cmd
$ cd new-tandhur-version
$ npm install
$ npm run dev
```

Production Enviroments:

```cmd
$ cd new-tandhur-version
$ npm install
$ npm start
```

Run on

```cmd
localhost:3500
```
