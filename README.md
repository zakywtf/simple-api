# health-monitoring-v1

Dashboard Health Monitoring

# Requiments:

- Node.JS

# Installation:

Dev Enviroments:

```cmd
$ cd health-monitoring-v1
$ npm install
$ npm run dev
```

Production Enviroments:

```cmd
$ cd health-monitoring-v1
$ npm install
$ npm start
```

Run on

```cmd
localhost:3500
```
