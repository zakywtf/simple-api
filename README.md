# nodebase-backend

Node Base Backend

# Requiments:

- Node.JS

# Installation:

Dev Enviroments:

```cmd
$ cd nodebase-backend
$ npm install
$ npm run dev
```

Production Enviroments:

```cmd
$ cd nodebase-backend
$ npm install
$ npm start
```

Run on

```cmd
localhost:3500
```
